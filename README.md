# HopeForAll NGO Website

## Project Description
This is a responsive NGO website built using HTML, CSS, and JavaScript.
The project focuses on building a clean user interface and implementing
basic interactivity without using any frameworks.

The website represents a social cause organization working to provide
education to underprivileged children.

---

## Week 2 Work
- Created responsive static website
- Designed hero, about, mission, impact and contact sections
- Implemented clean layout using Flexbox
- Deployed website using GitHub Pages
